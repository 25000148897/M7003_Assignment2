# M7003_Assignment2
The right word: cognition and mediation in the case of LLMs and the writing style of psychology papers

This project investigates stylistic changes in psychology papers writing style before and after the widespread adoption of large language models (LLMs).

## Corpus

- Pre-LLM papers (n = 30)
- Post-LLM papers (n = 30)
- Declared-LLM papers (n = 10)

A subset of papers is included for reference. The full corpus is described in the essay submitted via Cortex.

## Methods

Texts were extracted from PDFs and analysed using Python.

Key metrics:
- Grammar errors per 1,000 words
- Mean surprisal (predictability)
- Lexical diversity (MTLD)
- Burstiness (sentence-length variability)
- Syntactic complexity
- Reference density
